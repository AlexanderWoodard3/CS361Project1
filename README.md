# CS361Project1
Deterministic Finite Automata

## Synopsis
```
java fa.dfa.DFADriver ./tests/<SomeFile>.txt
```

## Description
This java program models a deterministic finite automata that recognizes a language.

## Compiling and Using
```
javac fa/dfa/DFADriver.java
java fa.dfa.DFADriver ./tests/<SomeFile>.txt
```
